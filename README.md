Инструкция по запуску программной системы через командную строку Windows

1. Распаковать архив с проектом в удобную папку.

2. Открыть командную строку Windows.

3. Перейти в папку проекта. Например:

```cmd
cd C:\Users\User\Desktop\mining_disp
```

4. Создать виртуальное окружение:

```cmd
python -m venv venv
```

5. Активировать виртуальное окружение:

```cmd
venv\Scripts\activate
```

После активации в начале строки командной строки должно появиться:

```cmd
(venv)
```

6. Установить необходимые библиотеки:

```cmd
pip install -r requirements.txt
```

7. Запустить программную систему:

```cmd
python main.py
```

8. После запуска откроется главное окно программной системы мониторинга состояния горной техники.

9. Для завершения работы необходимо закрыть окно программы.
